## 数据分析

- 从1990年的世界青春期女性生育率地图可以清楚的看出，非洲、南美洲的大部分国家与中东地区的地图区块显示的是红色，而最高的生育率达到了218%。
而发达国家较多的欧洲、北美和澳洲等地区地图区块颜色明显更浅。
**因此我的猜想“青春期女性生育率与经济情况可能相关”得到了初步的证明。

接下来将维度转为时间。当地图所显示的年份由1990年逐渐向2017年转变时，各国颜色均慢慢变浅。**由1990年地图上的大片红色转变为2017年的大范围蓝色，直至2017年，世界地图上只有非洲中南部的几个国家生育率呈现为红色。**

**可以看出，世界各国的青春期女性生育率均是有随着时间的推移在逐年减少**
